## ルール確認
リバーシとは
- 2人対戦
- コマが白と黒があります
- 最後にコマが多い方が勝ち
- もしくは相手のコマを裏返し切ったら勝ち
- 自コマで相手のコマを挟んだらひっくり返る
- オセロでは8x8なので今回は8ｘ8のマス目を用いる
